# Projeto base Mini Curso JSF - Unipar

slides: http://www.slideshare.net/emmanuelnerisouza/desenvolvimento-baseado-em-componentes-com-jsf

# Requisitos
- Java 7
- Tomcat 8.0.21
- Maven 3

# Frameworks utilizados
- JSF 2.2
- Servlet 3.1
- Primefaces 5.1

# Utilização
- Baixar projeto;
- Rodar mvn clean install;
- Adicionar tomcat na IDE;
- Fazer deploy do war.
